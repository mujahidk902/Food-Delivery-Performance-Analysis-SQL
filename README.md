# Food Delivery Performance Analysis Using SQL

## Project Overview

This project analyzes food delivery operations to identify factors affecting delivery performance, rider efficiency, and customer experience.

## Business Problem

Food delivery companies face challenges such as:
- Late deliveries
- Traffic congestion
- Weather disruptions
- Festival demand spikes

The goal is to identify key factors impacting delivery time and provide actionable business insights.

## Dataset Information

- Total Records: 2,988 Orders
- Features:
  - Delivery Time
  - Traffic Density
  - Weather Conditions
  - Vehicle Type
  - City Type
  - Rider Ratings
  - Festival Status

## Data Cleaning

- Checked NULL values

- Handled missing festival records
- Standardized data

## Business Questions Solved

1. Which traffic condition causes maximum delay?
2. Which weather condition increases delivery time?
3. What is the impact of festivals on delivery performance?
4. Which city category has the highest delivery time?
5. What is the relationship between distance and delivery time?
6. Which Order Type Takes More Time?
7. Which is Top Peroforming Riders ?
8. Top Cities by Orders.
9. Monthly Trend.
10. City-wise Delivery Performance.

## Key Insights

- Jam traffic creates the highest delivery delays.
- Cloudy and Foggy weather increase delivery time.
- Festival periods lead to longer delivery times.
- Metropolitan cities generate the highest order volume.

## Business Recommendations

- Increase rider allocation during festivals.
- Introduce dynamic routing during traffic jams.
- Improve weather-based delivery planning.
- Optimize rider deployment by city category.
